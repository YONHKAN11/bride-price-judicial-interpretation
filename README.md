## 核心内容

### 司法解释解决了什么

1. **明确彩礼范围**：以婚姻为目的、依习俗给付的财物属于彩礼；节日礼物、日常消费性支出等不属于
2. **禁止借婚姻索取财物**：以彩礼为名借婚姻索取财物的，法院支持返还
3. **已登记+共同生活**：离婚时一般不返还，但共同生活时间短且彩礼额过高的，可酌情返还
4. **未登记但共同生活**：根据共同生活时间、孕育情况、彩礼使用等因素确定是否返还及比例
5. **诉讼主体**：婚约双方父母可作为共同原告或共同被告
